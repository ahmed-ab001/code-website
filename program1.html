Program 1

Develop the HTML page named as “Myfirstwebpage.html”. Add the following tags with relevant content.

  1. Set the title of the page as “My First Web Page”

  2. Within the body use the following tags:

    1. Moving text =“Basic HTML Tags”
    2. Different heading tags (h1 to h6)
    3. Paragraph
    4. Horizontal line
    5. Line Break
    6. Block Quote
    7. Pre tag
    8. Different Logical Style(< b>,<u >,< sub >,< sup > etc.)

CODE                

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
     <style>
        body{
            font-family: Arial, sans-serif;
            line-height: 1.6;
            margin: 0;
            padding: 20px;
        }
    </style>
</head>
<body>
       <marquee>Basic HTML Tags</marquee>
       <h1>This is Heading 1.</h1>
       <h2>This is Heading 2.</h2>
       <h3>This is Heading 3.</h3>
       <h4>This is Heading 4.</h4>
       <h5>This is Heading 5.</h5>
       <h6>This is Heading 6.</h6>
       <p>This is a paragraph. It demonstrates the use of paragraph tag in HTML</p>
       <hr>
       <b>Bold Text</b><br>
       <i>Italic Text</i><br>
       <u>Underlined text</u><br>
       <strong>Strong Text</strong><br>
       <em>Emphasized Text</em><br>
       text with <sub>subscript</sub> and <sup>superscript</sup>
</body>
</html>
